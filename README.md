# 📱 WhatsApp Chat Analyzer

A modern **WhatsApp Chat Analyzer** built with **Python** and **Streamlit** that transforms exported WhatsApp chat files into meaningful insights through interactive dashboards, statistics, and visualizations.

The application helps users analyze chat activity, identify communication patterns, visualize trends, and explore word usage using Natural Language Processing (NLP) techniques.

# 📱 How to Export WhatsApp Chat

1. Open WhatsApp.
2. Open any chat.
3. Tap **⋮ (More Options)**.
4. Select **More → Export Chat**.
5. Choose **Without Media**.
6. Save the exported `.txt` file.
7. Upload it into the application.
